## Data Folder Architecture
```
data/
├─ Map1/
│   ├─ Map1.tif          ← your orthoimage (GeoTIFF or TIFF+Map1.tfw)
│   └─ Numbers/          ← one KML per field
│       ├─ 001.kml
│       ├─ 002.kml
│       └─ …
└─ Map2/
    ├─ Map2.tif
    └─ Numbers/
        ├─ 001.kml
        ├─ 002.kml
        └─ …

```
